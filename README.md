# Py_Input
---
개요
---
화면보호기를 해제가 불가능한 환경에서 화면보호기의 동작을 막기 위한 코드입니다.

---
1. 가장 먼저 Python을 설치를 해야 합니다. 
[Python 설치 링크](https://www.python.org/)

2. 콘솔창에 아래와 같은 Python 모듈을 설치해야 합니다.
```
> pip install pyautogui
```
3.실행 취소는 Ctrl-C누르면 실행이 자동으로 취소가 됩니다.
